## Description
NEST.js 를 활용한 공연 예매 사이트 API
<추가 사항>
 - 로그인 3회 시도 후 실패 시 30분 로그인 시도 금지 
 - 카드를 결제 수단으로 입력
 - 좌석에 등급 설정 가능(관리자 기능) 
 - 예매 및 결제 시 트랜잭션, 비관적 락
 - 예매 취소, 환불 시 트랜잭션
